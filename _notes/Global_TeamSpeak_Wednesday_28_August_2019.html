NOTES FOR THE HOST 
==================================================================================

Read this https://etherpad.net/p/How_to_hold_TZM_and_NHRM_meetings

Here is one way how to do it:
- Before starting the recording ask people to write their progress reports on meeting notes. Read what other people have written and ask them to add to the agenda anything they want to talk about (if there is a lot on agenda already, there is no need for this step)
- Start the recording
- Welcome people and remind them to use the speakers queue and agenda if they want to talk about something
    for example : Welcome to the the zeitgeist movement monthly wednesday meeting. It is 28 august 2019. Let's try to use speakers queue in the 
meeting notes. We usually try to keep these recorded meetings around one hour but everyone if welcome to the after party. If there is people on the queue, please try to keep your turn short. Something like two minutes. Holding meetings is a group effort so please try to make sure people follow rules.
- Start going through the agenda
   for example : First thing on our agenda is X. Who would like to tell us about it?
- When all the items in the agenda are done ask if there is anything important peope would like to talk on record.
- End the meeting. 
   for example : Thank you everybody for joining the meeting. Please stay for the afterparty if you have time. They tend to go on for hours.
- Stop recording
- Make sure you agree who will be hosting the next meeting. Please specify which weekday!
- later edit the recording and post it on archive.org. If you cannot edit it, find somebody who can do it.
